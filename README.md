# Editable Table

A beautiful, modern editable table built with vanilla HTML, CSS, and JavaScript.

## Features

- **Click to Edit**: Click any cell to start editing inline
- **Keyboard Navigation**: Press Enter to save, Escape to cancel
- **Add Rows**: Add new rows dynamically
- **Delete Rows**: Remove rows with smooth animations
- **Status Dropdown**: Select status from a dropdown menu
- **Export Data**: Export table data as JSON
- **Responsive Design**: Works on desktop and mobile devices
- **Dark Theme**: Modern dark UI with gradient accents

## Usage

Simply open `index.html` in a web browser, or serve the files using any static file server:

```bash
# Using Python
python -m http.server 8000

# Using Node.js (npx)
npx serve .
```

Then navigate to `http://localhost:8000`

## Technology

- Pure HTML5, CSS3, JavaScript (no frameworks)
- CSS Custom Properties for theming
- CSS Animations for smooth interactions
- Google Fonts (Outfit, JetBrains Mono)
